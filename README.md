# Análise de Vendas com Python

# Objetivo
Analisar dados de vendas para identificar padrões de faturamento, desempenho de produtos e regiões, auxiliando na tomada de decisão baseada em dados.

---

# Tecnologias utilizadas
- Python
- Pandas
- Matplotlib

---

# Estrutura do projeto
- data/ → base de dados
- notebook/ → análise em Python

---

# Análises realizadas
- Cálculo de faturamento total
- Identificação do produto mais vendido
- Identificação do produto com maior faturamento
- Análise de faturamento por região

---

# Insights obtidos
- A região com maior faturamento foi o Sudeste
- Produtos eletrônicos possuem maior impacto na receita
- O produto com maior faturamento foi o Celular
